## Rock_paper_scissor_test

A simple Rock-Paper-Scissors game built using Pygame. 

## Features
- Play against the computer
- Graphical buttons for choices
- Exit and Restart options

## Requirements
- Python 3.x
- Pygame

## How to run

Pygame_rock_paper_scissors.py



## Assets
- `rock.png`
- `paper.png`
- `scissors.png`
